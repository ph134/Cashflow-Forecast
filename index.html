User Guide: Input Sections

1. Opportunity Finance Overview

Enter opportunity details, contract value, currencies, FX rate, and manual margin.

2. Milestone Schedule

Add milestones, set percentages and invoice months, and confirm total milestone percent equals 100%.

3. Cost Elements

Enter each cost line, set currency pair and FX rate, and review converted costs.

4. Planned Expenditures

Set cumulative monthly progress by cost element to control cost timing.

5. Cashflow Estimate

Verify activity timing and review revenue/accrued totals.

6. Forecast

Review monthly revenue, cost, and cumulative cashflow values.

7. Cashflow Chart + Forecast Summary

Check the cash trend and summary metrics before export.
